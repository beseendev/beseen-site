# BeSeen - O seu talento merece ser visto

O **BeSeen** é uma rede social esportiva focada no ecossistema de futebol, conectando atletas a olheiros (scouts) e clubes profissionais. Esta é a Landing Page oficial do projeto, desenvolvida para transmitir profissionalismo, tecnologia e a energia do esporte.

## 🚀 Funcionalidades

- **Design Premium:** Interface em Dark Mode com estética inspirada no EA Sports FC e campanhas da Nike.
- **Jornadas Interativas:** Modais exclusivos para Atletas e Olheiros com animações que explicam o fluxo do app.
- **Efeitos Dinâmicos:**
  - Fundo animado (Atmosphere Field).
  - Radar de Scout (Scanner neon).
  - Cards 3D com efeito Parallax.
  - Animações de Scroll (Fade-in e Stagger).
- **Responsividade:** Totalmente adaptado para dispositivos móveis (Mobile First).
- **Páginas Auxiliares:** Inclui Central de Suporte e Política de Privacidade integradas.

## 🛠️ Tecnologias Utilizadas

- **HTML5 / CSS3** (Vanilla + Tailwind CSS via CDN)
- **JavaScript** (Interações de modal, animações de scroll e lógica de etapas)
- **Google Fonts** (Poppins e Inter)
- **Lucide/Custom SVG Icons**

## 📂 Estrutura do Projeto

```text
├── index.html       # Página principal (Landing Page)
├── privacy.html     # Política de Privacidade
├── support.html     # Central de Ajuda e Suporte
├── css/
│   └── style.css    # Estilos centralizados e animações
└── js/
    └── main.js      # Lógica de interatividade e modais
```
